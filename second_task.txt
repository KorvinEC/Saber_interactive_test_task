Пусть таблица (id, name, status, timestamp) называется First_table

1. Создаю новую таблицу Names
2. Переношу уникальные названия (name) из таблицы First_table в новую таблицу Names.
3. Добавляю новое поле (name_id) в таблицу First_table, с добавлением нужного id из таблицы Names
4. На стороне сервиса A необходимо произвести изменение,
либо чтобы присылался новый набор данных (id, name, name_id, status, timestamp),
либо создание нужной процедуры, которая будет выполнять поиск нужного name_id по name, и после этого будет производиться insert данных,
либо insert с встроеным select.
5. После этого возможно изменение сервиса Б, для того чтобы он забирал нужные данные либо с помощью select c join,
либо с помощью процедуры,
либо несколько запросов со стороны сервиса Б.
6. На данный момент сервис А и Б присылают и получают данные в формате (id, name_id, status, timestamp),
по этой причине мы можем удалить колонку name в таблице First_table.
